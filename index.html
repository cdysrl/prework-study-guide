HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • User visible elements include:
      1. Ordered Lists (Numbered List).
      2. Unordered Lists (Bullet Points).
      3. Text blocks.
  • HTML is the "house foundation and framing" of web page development.

CSS

Inline CSS

  • Styles directy in the HTML file.
  • Applies to a specific HTML tag.

Internal CSS

  • Added in the HTML < head > using < style > tag..

External CSS

Preferred Method
  • Styles stored in an external file.
  • Link the file using < link > in HTML < head >

Git

  • git status: checks what branch of repo we are currently on
  • git checkout -b branch-name: creates a new branch and moves to it
  • git checkout branch-name: move to that branch
  • git pull origin branch-name:
    • pull = download
    • origin = github repo
    THIS IS WHERE YOU START TO SYNC TO GITHUB!
  • git add -A:
    • -A = all files in directory
  • git commit -m "description":
    • commit = save
    • -m = adds a comment to commit, comment is typed in place of "description"
  • git pull origin main
  • git push origin branch-name
    • pull = upload
    • origin = github repo

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a variable which contain multiple variables (can have different data types)
  • Functions are stored procedures that you can 'call' throughout your code by using the function_name()
